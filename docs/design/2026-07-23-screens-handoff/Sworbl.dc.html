1sworbl × soomer — brand-aligned profile / leaderboard / settings
My read: sworbl is already most of the way to soomer — same ink-black canvas, same lavender accent, the same wry voice. So this tightens the family resemblance rather than reskinning. Deliberately kept as sworbl’s own: the candy tile palette and Fredoka numerals — that’s the game’s identity; going Manrope-only would make it read like a generic soomer web page instead of a toy. Net: unmistakably a Soomer joint, still unmistakably a game.
